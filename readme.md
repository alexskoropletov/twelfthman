#### 'My Gallery' SPA for TwelfthMan 

Powered by:
- Laravel 5.5
- MySQL 5.7.20
- Bootstrap 3.3
- jQuery 3.2.1
- ImageMagick 6.8.9-9

##### Database config:
- create .env file in root folder of the project
- copy .env.example
- set your database values

##### Migrations:
- php artisan migrate

##### Database seeding:
- php artisan db:seed --class=PhotosTableSeeder
